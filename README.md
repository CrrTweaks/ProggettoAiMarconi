# AI School Workspace

> **Production-ready AI-powered educational platform** combining smart school calendar, electronic register, teacher/student dashboards, local AI assistant (Ollama), PDF RAG, concept maps, voice support, and realtime chat.

## ✨ Features

- 🔐 **JWT Auth + RBAC** (Student / Teacher / Admin)
- 📅 **AI-powered Calendar** with workload balancing suggestions
- 📚 **Electronic Register** (homework, lessons, absences, exams)
- 🤖 **Local AI Assistant** via Ollama (llama3 / mistral / deepseek / codellama)
- 📄 **PDF RAG** with pgvector semantic search
- 🧠 **Concept Maps** generated by AI (React Flow)
- 🎙️ **Voice AI** (speech-to-text + text-to-speech)
- ⚡ **Realtime Chat & Notifications** via Socket.io
- 👑 **Omniscient Admin Dashboard** — visione globale multi-classe per presidi e segreteria
- 🎨 **Premium SaaS UI** — dark mode, glassmorphism, Framer Motion

## 🏗️ Architecture

```
┌────────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  React Vite    │────▶│  Node.js Express │────▶│  FastAPI AI      │
│  (Frontend)    │     │  (REST + WS)     │     │  (Ollama+RAG)    │
└────────────────┘     └────────┬─────────┘     └────────┬─────────┘
                                │                         │
                                ▼                         ▼z
                       ┌────────────────────────────────────┐
                       │  Supabase PostgreSQL + pgvector    │
                       └────────────────────────────────────┘
```

- **Frontend** → React 18 + Vite + Tailwind + shadcn/ui + Framer Motion + Zustand + TanStack Query
- **Backend Node.js** → Express ESM + JWT + bcrypt + Socket.io + Supabase SDK
- **Backend AI** → FastAPI + Ollama + LangChain + pgvector + PyPDF2 + speech_recognition + gTTS
- **Database** → Supabase PostgreSQL with pgvector extension

## 🚀 Quick Start

### Prerequisites

- **Node.js 20+**
- **Python 3.11+**
- **Docker & Docker Compose**
- **Ollama** running locally → https://ollama.ai
- **Supabase project** (or local Postgres with pgvector)

### 1. Clone & Configure

```bash
git clone <this-repo> ai-school-workspace
cd ai-school-workspace
cp .env.example .env
# Edit .env with your Supabase keys, JWT secret, Ollama host
```

### 2. Pull Ollama Models

```bash
ollama pull llama3
ollama pull mistral
# optional
ollama pull deepseek-coder
ollama pull codellama
```

### 3. Setup Supabase Database

Run the migration in the Supabase SQL editor (or via `psql`):

```bash
psql $DATABASE_URL -f database/schema.sql
psql $DATABASE_URL -f database/seed.sql
```

### 4. Start Everything (Docker)

```bash
docker compose up --build
```

Services:

- Frontend → http://localhost:5173
- Node API → http://localhost:4000
- AI API → http://localhost:8000

### 5. Start Manually (Dev mode)

```bash
# Terminal 1 - Node backend
cd backend-node && npm install && npm run dev

# Terminal 2 - AI backend
cd backend-ai && pip install -r requirements.txt && uvicorn app.main:app --reload --port 8000

# Terminal 3 - Frontend
cd frontend && npm install && npm run dev
```

## 🧩 API Surface

### Node.js (`:4000`)

- `POST /auth/register` · `POST /auth/login` · `POST /auth/refresh` · `POST /auth/logout`
- `GET/PUT /users/me`
- `GET/POST/PUT/DELETE /classes`
- `GET/POST /calendar/events`
- `GET/POST/PUT/DELETE /homework`
- `GET/POST /lessons`
- `GET/POST /absences`
- `GET/POST /exams`
- `GET/POST /interrogations`
- `GET /notifications` (+ Socket.io `notification:new`)

### FastAPI (`:8000`)

- `POST /chat` · `POST /chat/stream` (SSE)
- `POST /rag/upload` · `POST /rag/query`
- `POST /concept-map`
- `POST /voice/transcribe` · `POST /voice/tts`
